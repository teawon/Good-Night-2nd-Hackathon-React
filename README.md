### url

```
- / : 영화 목록 페이지
- /rated-movies : 페이지네이션 + 평가된 영화 목록 페이지
- /movie/{id} : 영화 상세 정보 페이지
- /create-movie : 영화 등록
- /edit-movie/{id} : 영화 수정
```
